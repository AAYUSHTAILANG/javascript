JAVA SCRIPT

                1 Object-based language
                2 <script>
                3 For Dynamic HTML

                Variables in JS
                ---------------
                1 var (available in function entirely)
                2 let (scope limited to block)
                3 const (once initiated then cannot be assigned again)
                * If no declaration is given then treated as a global variable.

                Operators in JS
                ----------------
                1 if x=y: assignment
                2 if x==y: compares value
                   ex. if x=10 and y="10"
                   (x==y) => TRUE
                3 if x===y: check datatype as well
                   (x===y) => FALSE

In JS, strings are allowed in

                1 " "
                2 ' '
                3 ` ` : advantage can take var value

Mode of getting JS into HTML

                1 External
                2 Internal
                3 Inline
                * If all are written then whichever is written first will be executed first.

Limitations of Document.write()

                * It erases the preexisting content of the page as the page gets reloaded on click of the button
                and inserts the new content only.

                Solution: - Use getElementById()
